# 📚 MyLib Frontend

**MyLib-Service의 프론트엔드 프로젝트**입니다.  
React, TypeScript, Vite를 기반으로 개발되었습니다.

## 🧩 기술 스택 (Tech Stack)

- **Framework**: React + TypeScript + Vite
- **상태 관리**: Zustand
- **스타일링**: TailwindCSS
- **빌드 도구**: Vite
- **코드 품질 관리**: ESLint + Prettier

## 💡 프로젝트 소개

MyLib은 사용자 맞춤형 도서 관리 및 상호작용 서비스를 목표로 하는 프로젝트입니다.
